Variables and Some Arithmetic

Compute the sum of squares for two integers using Python variables and arithmetic operators.

Question

Given two positive integers a and b, calculate a^2 + b^2.

a = 3
b = 4

Core idea

Use variables to store the input values, square each value with Python's exponent operator, then add the results.

Answer 1

a = 3
b = 4
print((a ** 2) + (b ** 2))

Output

25

Explanation

The expression a ** 2 squares a, and b ** 2 squares b. With a = 3 and b = 4, the program computes 9 + 16 and prints 25.
