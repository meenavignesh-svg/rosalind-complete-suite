Extract two inclusive slices from a string and print them with a space between them.

Question

Given a string s and four integers a, b, c, and d, return s[a] through s[b] and s[c] through s[d], inclusively.

a, b, c, d = 43, 47, 167, 174

Answer 1

s = "4Dt0IdYSzQOk1yltD8FfbCfGHaFZEOIecHXwIrTYriISorexHiyPRcdrFLWwN7Bi0fVK0aqgylLGO52ZU67ivxzIvuvcUVxe2ATqKeHskwfx1MTLtfQWIZZj3ugC8CiQRejz6ne05iwcfj9C3p0gK6U4bX0kwFWJySngI1tkorschunVy8fXf43Vs."
a, b, c, d = 43, 47, 167, 174
print(f"{s[a:b+1]} {s[c:d+1]}")

Code Explanation

The first slice is s[43:48], which returns Sorex. The second slice is s[167:175], which returns korschun. The formatted string joins them with one space.

Output

Sorex korschun

Performance Metrics

O(k) Python copies the two returned slices; k is the total returned character count.
